# 🎨 JS Color Palette Generator

A simple and responsive **Color Palette Generator** built with **vanilla JavaScript**.  
Generate random color palettes, copy HEX values in one click, and use it as quick inspiration for UI/design or frontend practice.

---

## ✨ Features

- ✅ Generate random color palettes
- ✅ Displays HEX color codes
- ✅ One-click copy to clipboard
- ✅ Clean, responsive layout (desktop + mobile)
- ✅ Built with pure HTML/CSS/JS (no libraries)

---

## 🧠 What I practiced

- DOM manipulation
- Event listeners
- Working with colors (HEX)
- CSS Grid / responsive design
- Clipboard API (copy feature)

1. Download the project as a ZIP file from GitHub.
2. Extract the folder on your computer.
3. Open `index.html` in your browser.
